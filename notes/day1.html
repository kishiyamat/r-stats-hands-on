2.2.1 中心や位置を表す指標 (p.15~)

平均値を \(\bar{X}=1/n\Sigma_{i=1}^{n}X_i\) としています。 人によっては下のRコードで見たほうが理解しやすいかもしれません。 具体的に、X_A_bar(つまり\(\bar{X}_\rm{A}\)) は以下のように求められます。

X_A_bar = sum(A)/length(A)
X_A_bar
## [1] 38.23714
# 課題: `X_B_bar` を求めてみてください。

中央値や最頻値は飛ばします。なお、繰り返しになりますが 母平均は\(\mu\)、標本平均は\(\bar{X}\)としています。

2.2.2 ばらつきを表す指標(a. 平方和)

X_A = df2[df2$labels=="SchoolA", "values"]  # 実データ
dev = X_A - mean(X_A)  # 偏差
S = (X_A - mean(X_A))^2  # squared
SS = sum(S)  # 平方和(2.2)
par(mfrow=c(1,3)) 
plot(X_A)  # 実データ
plot(dev)  # 偏差: 和は打ち消し合って0 -> all.equal(sum(dev), 0) 
plot(S)  # 平均で0、外れるほど+

2.2.2 ばらつきを表す指標(b.c)

    1. 標本分散SVは標本数で増えるのでnや(n-1)で割る
    1. 標準偏差\(\sqrt{s^2}\) (\(\sigma\)ではない)
  • 不偏分散\(s^2\) は(n-1)で割る コラム2.3を参照
SS = sum(S)  # 平方和(2.2)
n = length(X_A)
SV = SS/(n-1)  # 標本分散
sqrt(SV)  # 標準偏差
## [1] 2.593256
sd(X_A) == sqrt(sum((X_A - mean(X_A))^2)/(n-1))  # Rのsdはn-1バージョン
## [1] TRUE

3.1 大数の法則

p.28まで

大数の法則: 標本サイズが大きくなると平均は母集団平均に近づく

  • 以下の2つを比べたとき、大抵の場合は後者のほうが0.5(母集団平均)に近い
    • 10回のコイントスで平均を求める
    • 10000回のコイントスで平均を求める
  • 回数を大きくすると、0.5(母集団平均)に近づくというシミュレーションをしている。

3.2 正規分布

定義自体は(3.6)だが以下の性質が大事

  • 図3.4が示す通り、平均と標準偏差によって形が変わる
  • 先の下側の面積は1になる
  • 平均+-2x標準偏差の範囲が約0.95
  • 製造した缶の重さなどなど、多くのデータが従う
mean = 10
sd = 2
# 平均から2sdに収まる確率は、mean+2*sd から mean-2*sd を引けば良い
# pnormで与えたmeanとsdの分布におけるxまでの面積を求めくれる
# pnorm(-Inf, mean=mean, sd=sd)  # ->0
# pnorm(Inf, mean=mean, sd=sd)  # ->1
pnorm(mean+2*sd, mean=mean, sd=sd)-pnorm(mean-2*sd, mean=mean, sd=sd)
## [1] 0.9544997

3.3 中心極限定理

  1. とある特性(平均\(\mu\)としましょう)を持った母集団を用意します。
  2. 母集団から(例えば)10のデータを取り平均\(X_\rm{A}\)を求め、これを一試行とします。
  3. 上の試行をたくさん(教科書でいうところの\(M\)回)して\(X_\rm{A, B, ...}\)すると、
    • \(X_\rm{m}\)は\(\mu\)を平均とする正規分布に従います。
    • 一試行のサンプル数(上だと10)が小さいと収束が悪くなります。

-> これが「平均が集まって作られる分布」という概念です。 データが正規分布に従っているかは qqnormを使って見る方法や、 統計的に検証する方法があります(後述)。

Ch.4 推定と誤差

Ch.4 の目標

  • 手元のデータに基づいて、真の分布の平均値が95%の確率で入る範囲を知りたい
    • 例えば、真の分布の平均値は95%の確率で130–135cmに入る、など。
    • → 129や136は5%未満の確率→統計的に異なるグループです…と主張できる
  • そのためには、平均の分布の標準偏差（標準誤差）が必要
  • \(t\)検定に触れてみる

標準誤差

標本平均の分布の標準偏差を「標準誤差\(s_{\bar{X}}\)」と言います。式(4.1)で定義されます(p.37)。

  • nが大きいほど小さくなる -> 図4.2
  • 標本標準偏差\(s\)が小さいほど小さくなる -> 図4.3 (cf.図4.2)
d = rnorm(2000, mean=145.5, sd=4.0)  # 母集団
hist(d)

4.2 正規分布とt分布

  • 式(4.2)の\(u\)は定義から以下の性質があります。
    • \(u\)は標本平均が母平均\(\mu\)と同じ時に0になる
    • \(u\)は標本平均が標準偏差と同じ時に1になる
    • 標準正規分布N(0,1)に従う
    • → 「155cmというデータがある\(\mu\)（例えば145.5）の分布から出てきた、仮説を提案します。 でも、その確率は5%未満みたいでう。起こりそうもないので棄却します」 という論法が使える。
  • しかし現実に(4.2)にある\(\sigma^2\)は不明なので、 代わりに\(s^2\)を使って\(u\)ではなく\(t\)を定義します。
    • 分母は標準誤差\(s_{\bar{X}}\)です。
    • \(t\)は標準正規分布ではなく、よく似た\(t\)分布に従います。 「自由度」によって多少、形は変わります。
    • → おなじ論法が使える

注意: 式(4.2)は\(t\)分布分布の式ではありません。\(t\)分布に従う値、 \(t\)値の式です。式(4.2)が正規分布の式(3.6)でないのと同じことです。

4.3 95%信頼区間と1標本のt検定

  • (4.3)を変形すると(4.4)になります。
    • \(t_{(n-1, \alpha/2)}\) とは \(n-1\)、つまり「自由度」に従う\(t\)分布を用意した際、 0.05/2、つまり左から0.025%を含む\(t\)を教えて、という意味
    • 図4.4を見ると、たしかに-2.7は左端になっている。
    • (4.4)の範囲に\(\mu\)は95%の確率で入る （図4.5は仮想的に20回の実験をして、 95%信頼区間に平均が19回入ることを示している。 あるグループが枠の外ならば、別のグループのデータだと主張できそう）

演習問題4.1

(2)の答え合わせ

# 4.1: 式(4.1)
data = c(15.2, 11.3, 15.7, 18.2, 12.4, 13.0, 16.6, 17.3, 16.2, 18.7, 14.5, 13.7)
# (1)
n = length(data)
s_x = sd(data)/sqrt(n)
# (2)
p=0.05
X_bar = mean(data)
cred_low = X_bar+qt(p/2, n-1)*s_x
cred_high = X_bar+qt(1-p/2, n-1)*s_x
print(c(cred_low, cred_high))
## [1] 13.75579 16.71088

演習問題4.2

いえる

cred_low = 13.5+qt(p/2, n-1)*s_x
